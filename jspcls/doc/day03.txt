# day03

1. 서버가 응답문서를 응답하는 방식
	1) forward 방식
		==> 요청내용 (url + 데이터)은 그대로 두고
			응답해주는 문서만 바꿔서 보여주는 방식
	
	2) redirect 방식
		==> 클라이언트가 요청한 내용을 모두 지우고
			서버에게 새롭게 요청을 하는 방식
			
2. 요청설계
	==> 클라이언트가 요청하는 내용과 입력해야 할 데이터
		해당 요청에서 실행되어야 할 클래스
		그리고 요청으로 응답받아야 할 뷰 파일을 정리해 놓은 문서

작성 문서의 형식은 따로 정해져있지 않고
필요한 항목이 누락없이 기입되어 있으면 된다.

필요항목 ]
	
	요청내용 : 요청하는 경로 (url)
	파라미터 : 요청시 입력해야할 데이터
	컨트롤러 : 해당 요청에서 실행되어야 할 자바 클래스
	뷰 : 응답해줘야 할 응답 문서
